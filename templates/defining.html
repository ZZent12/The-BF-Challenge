Seperating the Problems

We can split the objective into three parts. The first part is actually creating the bf program. The second part is what I call the decoding process. This would be the part where we calculate the values for each cell and find out what cells we are going to output. The third and final process is what I call parse. This part will convert the values of the outputted cells into their corresponding letter.

Solving the Problems

Part 2 - Creating the Decode process

To start off we should do the more basic things first like creating the pointer, tape, and getting the basic commands to work as intended.

  • def Decode(file):
  •     Tape = [0 for _ in range(100)]
  •     Pointer = 0
  •     with open(file, "r") as f:
  •         Script = f.read()
  •     for Command in Script:
  •         if Command == "<":
  •             Pointer -= 1
  •         elif Command == ">":
  •             Pointer += 1
  •         elif Command == "+":
  •             Tape[Pointer] += 1
  •         elif Command == "-":
  •             Tape[Pointer] -= 1

This is the code that I came up with to handle the basic commands. It will undoubtedly change as we add in the complex commands, but let me quickly explain what we have so far. It starts with "def Decode(file):", all this does is create a function with the name of "Decode" and it takes an argument (which is a value or values that are use help determine the behavior of a program) that is stored in a variable called file. The next line creates a variable called tape and it is a list of values that are all set to the value 0. The third line will create a variable called Pointer and set its value to 0. We can use these 2 variables together to make the basic commands of Brainfuck. This is achievable because the list data type is an ordered collection of values that allows duplicates and we can get the value of one specific element (item) by telling what number item it is in the list. So if we want to move to the right then we need to add 1 to the current value of the pointer and if we want to move to the left we would subtract 1 from the pointer's value. Then to add or subtract to the cell's value we need to add or subtract 1 from the item that is at the pointer's value.
